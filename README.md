## Description

Site de service pour la Clinique Sourire Plus, une clinique dentaire fictive à Ottawa. Le site permet aux utilisateurs de :

- Se renseigner sur les services offerts (nettoyage, blanchiment, examen complet)
- Comprendre le traitement de canal et son coût estimé
- Prendre un rendez-vous en ligne en 3 étapes
- Recevoir une confirmation par courriel automatique

## Personas

- **Jahir Hassan** : veut comprendre le traitement de canal et son coût avant de prendre rendez-vous
- **Sophie Martin** : veut prendre un rendez-vous pour un nettoyage directement depuis la page d'accueil

## Technologies

- React 19 (Create React App)
- CSS personnalisé — thème noir et or
- EmailJS — confirmation par courriel
- GitHub Pages — hébergement
